# python_first_api

### steps:

- init venv
- `pip install django`
- `django-admin startproject <name_project>`
- `python ./manage.py migrate`
- `python ./manage.py runserver`
- `python ./manage.py startapp <name_module>`
- add new module in `settings` -> `INSTALLED_APPS`
